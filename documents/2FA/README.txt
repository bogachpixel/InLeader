Инструкция по настройке 2FA для бота

Файлы:
- 2FA.pdf или 2FA.txt — текст инструкции. Разбивайте на шаги форматом «Шаг 1:», «Шаг 2:» и т.д.
- step1.png, step2.png, ... — изображения для каждого шага (скриншоты, схемы). Необязательно.

Бот читает файл, разбивает по шагам и показывает по одному. Пользователь должен нажать «Выполнил» перед переходом к следующему шагу.
В конце показывается поздравление.
